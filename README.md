# Carey Barry — Personal Portfolio

A personal portfolio website for Carey Barry, PhD student in Personal Health Informatics at Northeastern University, clinical professor, and surgical PA. The site highlights her academic background, publications, teaching, and campus photography.

---

## Project Objective

Build a clean, accessible, multi-page personal portfolio website to showcase professional accomplishments, research publications, teaching experience, and personal interests — developed as part of the Web Development course at Northeastern University.

## Tech Requirements

| Tool / Technology | Purpose |
|---|---|
| HTML5 | Page structure and content |
| CSS3 | Custom styling and layout |
| [Bootstrap 5.3](https://getbootstrap.com/) | Responsive grid and components |
| [ESLint](https://eslint.org/) | JavaScript linting |
| [Prettier](https://prettier.io/) | Code formatting |
| [http-server](https://www.npmjs.com/package/http-server) | Local development server |
| [reload](https://www.npmjs.com/package/reload) | Live reload during development |
| Node.js / npm | Dependency management |

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npx http-server
```
